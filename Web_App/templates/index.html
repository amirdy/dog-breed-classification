Using pretrained EfficientNet-B3 model

Welcome
This Application predicts the dog breed from 150 different breeds ( Accuracy ~ 92.6 % ).
  1. Select a random image or upload your dog image.
  2. Click the Run button to get the result.
Note:

If you upload any image and then click the Run button, the image will be removed from the server.

Notice

We have included 150 dog breeds in this program despite there being more than 150 breeds in the world; therefore, it is possible for your picture to be amongst those that we have not included. In this case, the program suggests the closest breed that resembles your picture. For example, American Eskimo Dog is not included in the 150 considered breeds. In this case, the program will suggest Samoyed which resembles American Eskimo Dog the most.
This application has the highest accuracy for the 150 considered breeds (see the 150 breeds).
